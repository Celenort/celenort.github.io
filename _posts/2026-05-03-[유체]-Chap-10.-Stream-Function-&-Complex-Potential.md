# Chap 10. Stream function and complex potential


## 10.1 Stream Function


Continuity equation


$\nabla\cdot\vec{u}=0$


by 


{% raw %}
$$
\frac{\partial u}{\partial x}+\frac{\partial v}{\partial y}=0
$$
{% endraw %}



Let 


{% raw %}
$$
u=\frac{\partial\psi}{\partial y},\quad v=-\frac{\partial\psi}{\partial x}
$$
{% endraw %}



to make $\psi$ satisfy continuity equation


## 10.2 Complex Potential


In a complex domain, define a function such that


{% raw %}
$$
f(z)=\phi(x,y)+i\psi(x,y)
$$
{% endraw %}



If $f(z)$ is analytic function (differentiable for $\forall z$) the necessary condition is


{% raw %}
$$
\frac{\partial\phi}{\partial x}=\frac{\partial\psi}{\partial y},\quad\frac{\partial\psi}{\partial x}=-\frac{\partial\phi}{\partial y}
$$
{% endraw %}



So-called Cauchy-Riemann condition


$z=x+iy$, $\frac{\partial z}{\partial x}=1,\quad\frac{\partial z}{\partial y}=i$


{% raw %}
$$
\frac{\partial f}{\partial x}=\frac{\partial\phi}{\partial x}+i\frac{\partial\psi}{\partial x}
$$
{% endraw %}



{% raw %}
$$
=\frac{\partial z}{\partial x}\left(\frac{\partial\phi}{\partial z}+i\frac{\partial\psi}{\partial z}\right)\to\frac{\partial f}{\partial z}=\frac{\partial\phi}{\partial x}+i\frac{\partial\psi}{\partial x}
$$
{% endraw %}



{% raw %}
$$
\frac{\partial f}{\partial y}=\frac{\partial\phi}{\partial y}+i\frac{\partial\psi}{\partial y}=\frac{\partial z}{\partial y}\frac{\partial f}{\partial z}
$$
{% endraw %}



{% raw %}
$$
\frac{\partial f}{\partial z}=\frac{\partial\psi}{\partial y}+i\frac{\partial\phi}{\partial x}
$$
{% endraw %}



{% raw %}
$$
\to\frac{\partial\phi}{\partial x}=-\frac{\partial\psi}{\partial y},\quad\frac{\partial\psi}{\partial x}=\frac{\partial\phi}{\partial y}
$$
{% endraw %}



Differentiate with $x$,


{% raw %}
$$
\frac{\partial^2\phi}{\partial x^2}=\frac{\partial}{\partial x}\left(\frac{\partial\psi}{\partial y}\right)=\frac{\partial}{\partial y}\left(-\frac{\partial\psi}{\partial x}\right)=-\frac{\partial^2\phi}{\partial y^2}
$$
{% endraw %}



{% raw %}
$$
\frac{\partial^2\phi}{\partial x^2}+\frac{\partial^2\phi}{\partial y^2}=0
$$
{% endraw %}



$\nabla^2\phi=0$


Similarly, $\nabla^2\psi=0$


{% raw %}
$$
f(z)=\phi(x,y)+i\psi(x,y)
$$
{% endraw %}



{% raw %}
$$
u=\frac{\partial\phi}{\partial x}=\frac{\partial\psi}{\partial y}
$$
{% endraw %}



{% raw %}
$$
v=\frac{\partial\phi}{\partial y}=-\frac{\partial\psi}{\partial x}
$$
{% endraw %}



{% raw %}
$$
\Rightarrow \Phi(x,y)=\phi(x,y)+i\psi(x,y)
$$
{% endraw %}



## 10.3 Streamline


{% raw %}
$$
\frac{dx}{u}=\frac{dy}{v} \quad \text{ or }\quad v\,dx-u\,dy=0
$$
{% endraw %}



Value of stream function is constant through streamline


{% raw %}
$$
u=\frac{\partial\psi}{\partial y},\quad v=-\frac{\partial\psi}{\partial x}
$$
{% endraw %}



{% raw %}
$$
\frac{\partial\psi}{\partial x}dx+\frac{\partial\psi}{\partial y}dy=d\psi=0
$$
{% endraw %}



All velocities are tangent to the line


![](/assets/img/2026-05-03-[유체]-Chap-10.-Stream-Function-&-Complex-Potential/0-1774cdd7bd.png)


{% raw %}
$$
\frac{\phi_2-\phi_1}{\delta x}=u
$$
{% endraw %}



{% raw %}
$$
u=\frac{\psi_1-\psi_2}{\delta y}
$$
{% endraw %}
